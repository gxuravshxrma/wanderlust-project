# JourneyQuest – Your Ultimate Travel Experience

Welcome to **JourneyQuest**, a travel website designed to help you explore and plan your next great adventure! This project demonstrates how HTML, CSS, PHP, and JavaScript come together to create a functional, user-friendly platform.

---

## Overview

**JourneyQuest** lets users discover various travel destinations, book trips, and get in touch with travel experts. This project showcases how basic web development technologies can be used to build a responsive travel website.

---

## Features

- **Explore Destinations**: View various travel destinations with detailed descriptions and images.
- **Book Your Journey**: Choose and book your next adventure.
- **Get in Touch**: Contact travel experts for personalized recommendations.
- **Responsive Design**: The website is mobile-friendly and works on desktop, tablet, and mobile devices.

---

## Technologies Used

- **HTML5**: Website structure and content.
- **CSS3**: Styling for an engaging and responsive layout.
- **PHP**: Backend functionality to manage form submissions and database interactions.
- **JavaScript**: Added interactivity to enhance the user experience.

---

## Project Structure

- **index.html**: The main homepage of the website.
- **contact_us.php**: Backend script for handling form submissions.
- **css/**: Folder containing all the styling files.
- **js/**: Folder for JavaScript files to manage dynamic content.
- **img/**: Folder for images and media assets used throughout the website.
- **db_config.php**: Configuration file for database connections (used by PHP scripts).
